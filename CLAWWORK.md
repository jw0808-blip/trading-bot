# CLAWWORK.md - TraderJoes Firm Agent Framework

Reference this file for all agent and sub-bot development.

## Core Principles
- Agents must have defined skills and tools.
- Every agent logs corrections to learnings/LEARNINGS.md.
- Use structured output (markdown, emojis, bold).
- Self-improving loop: mistake → log → update rules.
- Never expose keys.
- Follow SECURITY.md and CLAUDE.md at all times.
- Only build agents that create real leverage.

## Key Skills for Our Agents
- Research
- EV Calculation
- Liquidity Scan
- Geopolitical Analysis
- Technical Analysis (TradingView)
- Risk Management
- Code Generation & Debugging
